drawing_program.

A simple drawing tool created using Java.



Using Java, I create a simple drawing program. Several brushes are implemented in the program, giving users the feel of traditional art: the pencil, calligraphy pen, and the airbrush, which can be switched with the press of a button, gives the user a variety of different drawing styles. The color can also be changed, and the program also comes with saving and recording capabilities.


You may need to have some permissions, such as JavaScript, enabled.

CONTROLS

[1] - [4]
Change Brush

[5] - [8]
Eraser

[Q] - [P] (top row alphabet keys)
Change Color

[A], then [SPACE]
Reset canvas

[S]
Screenshot canvas*

[D], then [SPACE]
Record canvas*

*Note: Saving features such as screenshots and recording are not functional in this web version. However, I have decided to keep these features here for experimental reasons; they will break and sometimes create interesting unintended effects (try recording the canvas, then use the brush tools!)
